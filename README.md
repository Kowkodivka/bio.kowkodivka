# Overview

bio.kowkodivka is my personal website — a simple digital profile where I keep
the most important info about myself, useful links, and ways to reach me.

## Development & Build

To run the project locally:

```shell
bun install
bun dev
```

To build it for production:

```shell
bun build
```
